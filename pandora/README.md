# Description
Theme Pandora, it is only for linux system or windows subsystem linux :u

## Usage

Copy file `chenfetch` to `/usr/bin`
```
cp bin/chenfetch /usr/bin/
chmod +x /usr/bin/chenfetch
```

To usage you can run `chenfetch`
```
chenfetch
```
